## 📋 구매 절차 (5분)

### Step 1: 가비아 회원가입

1. [가비아](https://www.gabia.com) 접속
2. 우측 상단 **회원가입** 클릭
3. 정보 입력 (이메일, 비밀번호, 휴대폰 인증)
4. 회원가입 완료

---

### Step 2: 도메인 검색

1. 메인 페이지 검색창에 원하는 도메인 입력
   ```
   예: resonance
   ```

2. **검색** 클릭

3. 사용 가능한 도메인 확인
   ```
   ✅ resonance.com - 15,400원/년
   ✅ resonance.co.kr - 15,000원/년
   ✅ resonance.kr - 13,200원/년
   ✅ resonance.io - 48,400원/년
   ❌ resonance.net - 사용 불가 (이미 등록됨)
   ```

---

### Step 4: 장바구니 & 결제

1. 원하는 도메인 **장바구니 담기**
2. 장바구니 확인
3. **구매하기** 클릭

#### 옵션 설정
- **등록 기간**: 1년 (추천)
- **도메인 보호 서비스**: 선택 (무료)
- **후이즈 정보 보호**: 선택 (유료 2,000원/년) - 개인정보 보호용

#### 결제 방법
- 신용카드
- 계좌이체
- 무통장입금

---

### Step 5: 구매 완료

결제 완료 후:
- 이메일로 구매 확인 메일 수신
- **My가비아** → **서비스 관리** → **도메인**에서 확인 가능

## 🔗 Vercel과 연결하기

### Step 1: 가비아 DNS 관리 페이지

1. [My가비아](https://my.gabia.com) 로그인
2. **서비스 관리** → **도메인**
3. 구매한 도메인 클릭
4. **DNS 정보** → **DNS 관리** 클릭

---

### Step 2: DNS 레코드 추가

**Vercel 제공 정보**:
```
A Record: 76.76.21.21
CNAME: cname.vercel-dns.com
```

#### A 레코드 추가
```
레코드 타입: A
호스트: @
값/위치: 76.76.21.21
TTL: 3600 (1시간)
```

**추가** 클릭

#### CNAME 레코드 추가
```
레코드 타입: CNAME
호스트: www
값/위치: cname.vercel-dns.com
TTL: 3600
```

**추가** 클릭

---

### Step 3: 기존 레코드 삭제 (중요!)

가비아는 기본 레코드가 있습니다. **삭제해야 합니다!**

다음 레코드들을 찾아서 **삭제**:
- 기존 A 레코드 (211.x.x.x 형태)
- 기존 CNAME 레코드 (parking 관련)

---

### Step 4: 저장 및 대기

1. **설정 저장** 클릭
2. DNS 전파 대기: **10분~48시간**
   - 보통 **20-30분** 내 완료

---

## ✅ DNS 전파 확인

### 방법 1: 온라인 도구
1. [DNS Checker](https://dnschecker.org) 접속
2. 도메인 입력
3. 타입: A 선택
4. 전 세계 DNS 전파 상태 확인

### 방법 2: 명령어
```bash
# Windows
nslookup yourdomain.com

# Mac/Linux
dig yourdomain.com
```

## 🚨 자주 하는 실수

### 실수 1: 기존 레코드 삭제 안함
❌ **문제**: 도메인이 가비아 광고 페이지로 연결됨
✅ **해결**: 기존 A, CNAME 레코드 모두 삭제

### 실수 2: www 없이만 설정
❌ **문제**: www.yourdomain.com 접속 안 됨
✅ **해결**: CNAME 레코드 꼭 추가

### 실수 3: TTL 너무 길게 설정
❌ **문제**: DNS 변경이 느리게 반영
✅ **해결**: TTL을 3600 (1시간)으로 설정
