# Resonance Company - Official Website

고급 인플루언서 & 퍼포먼스 마케팅 에이전시 웹사이트

## 🎯 프로젝트 개요

Resonance Company(RSNC)는 인플루언서 마케팅과 퍼포먼스 마케팅을 전문으로 하는 에이전시입니다. 이 웹사이트는 브랜드의 핵심 철학인 "Selection over Noise"를 시각적으로 표현하며, 다크 테마와 네온 그린 액센트를 활용한 미니멀하고 세련된 디자인을 특징으로 합니다.

## 🚀 기술 스택

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: React Icons
- **Font**: Inter, Space Grotesk (Google Fonts)

## 📦 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 🔧 커스터마이징

### 색상 변경
`tailwind.config.js`에서 네온 그린 색상을 수정할 수 있습니다:

```js
colors: {
  neon: {
    green: '#7CFF00', // 원하는 색상으로 변경
  },
}
```

### 콘텐츠 수정
각 섹션 컴포넌트는 `components/sections/` 폴더에 있습니다.
